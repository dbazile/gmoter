# gmoter

![screenshot](/screenshot.png)

Just a one-file emoticon picker.  Bind it to a keyboard shortcut and
carry on.

Maybe this goes away one day when
[this](https://wiki.gnome.org/Design/OS/Emoji) becomes an official
Gnome feature. :)

## Requires

- Python 2
- PyGTK
